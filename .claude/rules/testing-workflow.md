# Kiểm thử / xem trước

Không có build step. Để xem trang:
- Mở trực tiếp `index.html` bằng trình duyệt, hoặc
- Chạy local server đơn giản (ví dụ `npx serve .` hoặc extension Live Server) để test
  các đường dẫn tương đối chính xác hơn.

Trước khi báo hoàn thành một thay đổi giao diện, mở trang trong trình duyệt để kiểm tra
thực tế (bố cục, responsive, các trạng thái hover/focus), không chỉ dựa vào đọc code.
